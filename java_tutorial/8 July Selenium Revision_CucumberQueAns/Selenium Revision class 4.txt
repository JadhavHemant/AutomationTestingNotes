How to handle alert Pop up?
	- We can' t inspect the  alert pop up objects.
	- if u want to handle these type of pop up then first we have switch focus to main window to alert pop up window by using switchTo().alert() method.
	- switchTo() method present inside the WebDriver interface and return type is TargetLocator interface.
	- alert() method present inside the TargetLocator interface and return type is Alert interface.
	- In Alert pop up we have 4 objects are available.
		1) ok button
		2) cancel button
		3) physical text 
		4) text box.

	- if u want to click on ok button then we use accept() method
	- if u want to click on cancel button then we use dismiss() method
	- if u want to capture pop up text then we use getText() method
	- if u want to enter text then we use sendKeys() method



Scenario:
	- open a https://demoqa.com/alerts
	- click on 1st click on button
	- capture alert physical text and click in ok button

		WebElement wb = driver.findElement(By.name(""));

step 4: click on it
		wb.click();

step 5: switch focus from main window to alert pop up
	Alert alt = driver.switchTo().alert();

step 6: capture physical text

step 7: click on ok button
	alt.accept();
