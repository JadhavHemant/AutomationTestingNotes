what is Synchronization?
		- it is process of matching speed of application under test and test tool to get proper execution flow then we Synchronization concept.


why we use Synchronization?
		during the test execution selenium give instruction one by one to web element with same speed but in application under test, it takes less time for some element and it takes more time for some elements to find the object in web page, in order to maintain the test execution flow we use Synchronization concept.

there are 2 types of Synchronization
		1) unconditional Synchronization
		2) conditional Synchronization

1) unconditional Synchronization
		-in unconditional Synchronization, we specify the time in ms, we will wait certain amount of time, once the time is completed then proceed to next steps. 

syntax;
			Thread.sleep(10000);


2) conditional Synchronization
		- conditional Synchronization it apply for the findElement() and findElements() method.
		- it does not work for all commands or statement in selenium.
		there are 3 types of conditional Synchronization
		1) implicitly Wait
		2) explicit wait
		3) fluent wait.

1) implicitly Wait
	- implicitly Wait is applicable for all the elements in web page.
	- implicitly Wait is global wait.
	- implicitly Wait is also called dynamic wait.
	- implicitly Wait in selenium used to tell WebDriver to wait certain amount of time before throwing the NoSuchElementException.
	- if we define implicitly Wait as 30 seconds and if element found within the 5 seconds then it ignore remaining all seconds.
	- if we define implicitly Wait as 30 seconds and if element is not found  within the 30 seconds then it throws NoSuchElementException.
	- implicitlyWait() method present inside the Timeouts interface and return type is Timeouts interface.

syntax:
		driver.manage().timeouts().implicitlyWait(Duration.ofSeconds(30));				---> as per the selenium 3 or 4 and above version.
		
		driver.manage().timeouts().implicitlyWait(30,TimeUnit.SECONDS);				as per the selenium 3 and below version.
		
	- manage() method present inside the WebDriver interface and return type of manage() method is Options interface.
	- timeouts() method present inside the Options interface and return type is TimeOuts interface.
	- implicitlyWait() method is present inside the Timeouts interface and return type is Timeouts interface.

------------------------------------------------------------------------------------------------------------------------------------------------
2) explicit wait
	- explicit wait it is applicable for single element in web page.
	- explicit wait is also called as local wait.
	- explicit wait is selenium used to tell WebDriver to wait certain condition or Expected Conditions or maximize time before throwing the
	TimeoutException.
	- if we define the explicit wait as 30 seconds and if element found within the 2 seconds then it will remaining all seconds.
	- if we define the explicit wait as 30 seconds and if element is not found within 30 seconds then it will throw TimeoutException.

	- we have to create Object of WebDriverWait class by passing driver instance and times in seconds.		
	- then we with the help of WebDriverWait instance we use until() by passing ExpectedConditions.
	- return type of until() method is WebElement interface.
				


syntax:
		WebElement wb = driver.findElement(By.id("id value"));

		WebDriverWait wait =new WebDriverWait(driver, Duration.ofSeconds(30));
		wait.until(ExpectedConditions.visibilityOf(wb));
package Tutorial13;

import java.time.Duration;

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.support.ui.ExpectedConditions;
import org.openqa.selenium.support.ui.WebDriverWait;

public class Demo1 {

	public static void main(String[] args) {
		WebDriver driver = new ChromeDriver();

		driver.get("https://www.facebook.com/reg");

		driver.manage().timeouts().implicitlyWait(Duration.ofSeconds(30));

		WebElement wb1 = driver.findElement(By.name("firstname"));
		wb1.sendKeys("Ajit");

		WebElement wb2 = driver.findElement(By.name("lastname"));
		wb2.sendKeys("Patil");

		WebElement wb3 = driver.findElement(By.name("reg_email__"));
		wb3.sendKeys("ajit@gmail.com");

		WebElement wb4 = driver.findElement(By.name("reg_passwd__"));

		WebDriverWait wait = new WebDriverWait(driver, Duration.ofSeconds(45));
		wait.until(ExpectedConditions.visibilityOf(wb4));

		wb4.sendKeys("Ajit@123");

	}

}
