how to handle dynamic table in selenium?
---------------------------------------------------------------------------------
	- first we locate the calendar object using findElement() method and then we click on it. 
	 - then once it open a calendar then we run infinite loop to find the expected month and year.
	- inside the loop we capture actual month and year using getText() method and we compare with expected month and year.
	- if it is equal then we break the loop. and if it is  not equal then we click on next button.
	- once we find expected month and year then we find all dates using findElements() method.
	and then we iterate all dates using loop and, inside the loop we capture actual dates and we compare with expected date.
	- if actual date is equal to expected date then we click on date and break.

---------------------------------------------------------------------------------
Scenario:
		- open a application
		- click on calendar icon
		- find and select the expected month and year as well as date.

step 1: up casting
			WebDriver driver = new ChromeDriver();

step 2: open a url
			driver.get("");

step 3: find a calendar icon or object or element
			WebElement wb = driver.findElement(By.name(""));

step 4: click on calendar object 
			wb.click();

step 5: run iterate infinite loops to find the expected month and year calendar

		while(true)
		{
				step 6: find month and year
				WebElement monthAndYear = driver.findElement(By.xpath(""));

				step 7: capture actual month and year
				String actualMonthAndYear =monthAndYear.getText();

				step 8: compare actual month and year with expected month and year
				if(monthAndYear.equalsIgnoreCase("Expected Month and year"))
				{
						//step 9: break the loop
					break;
				}
				else 
				{
						//step 10: if actual month and year is not equal to expected month and year then click on next button
						//find next button and click on it 
						WebElement nextButton = driver.findElement(By.xpath(""));
						nextButton.click();
				}
		}


step 11: find all dates 
			List<WebElement> listDates = driver.findElements(By.xpath(""));

step 12: iterate all dates
			for(WebElement date: listDates)
			{
				//step 13: capture actual dates
				String actualDate = date.getText();

				//step 14: compare actual date with expected date
				if(actualDate.equalsIgnoreCase("Expected date"))
				{
						//step 15: click on date
						date.click();
				}
			}

package Tutorial5;

import java.util.List;

import org.openqa.selenium.By;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WebElement;
import org.openqa.selenium.chrome.ChromeDriver;
import org.openqa.selenium.chrome.ChromeOptions;

public class Demo1 {

	public static void main(String[] args) throws InterruptedException {
		
		ChromeOptions opt = new ChromeOptions();
		opt.addArguments("--disable-notifications");
		
		// step 1: up casting
		WebDriver driver = new ChromeDriver(opt);
		driver.manage().window().maximize();
		// step 2: open a url
		driver.get("https://www.redbus.in/");

		// step 3: find the date object or icon and click it
		WebElement wb = driver.findElement(By.id("onwardCal"));
		wb.click();

		Thread.sleep(4000);
		// step 4: run infinite loop to find the expected month and year
		while (true) {
			Thread.sleep(2000);
			// step 5: find month and year and capture actual month and year
			String actualMonthAndYear = driver
					.findElement(By.xpath("(//div[@class='DayNavigator__IconBlock-qj8jdz-2 iZpveD'])[2]")).getText();

			// step 6: compare actual month and year with expected month and year
			if (actualMonthAndYear.contains("Mar 2025")) {
				// step 7: break the loop
				break;
			} else {
				// step 8: find next button and click on it
				driver.findElement(By.xpath("(//div[@class='DayNavigator__IconBlock-qj8jdz-2 iZpveD'])[3]")).click();
			}
		}

		// step 9: find all dates
		List<WebElement> listDates = driver.findElements(By.xpath(
				"//span[@class='DayTiles__CalendarDaysSpan-sc-1xum02u-1 dkWAbH' or @class='DayTiles__CalendarDaysSpan-sc-1xum02u-1 bwoYtA']"));

		// step 10: iterate all dates
		for (WebElement date : listDates) {
			// ste 12: capture actual date
			String actualDate = date.getText();

			// step 13: compare actual Date with expected date
			if (actualDate.equalsIgnoreCase("10")) {
				// step 14: click on date
				date.click();
				//step 15: break the loop
				break;
			}
		}

	}

}

		opt.addArguments("--disable-notifications");

		driver.get("https://www.goibibo.com/");

		Thread.sleep(7000);
		// step 3: find the date object or icon and click it

				"//span[text()='Departure']/following-sibling::p[@class ='sc-12foipm-4 czGBLf fswWidgetTitle']"));

		Thread.sleep(4000);

		}

		List<WebElement> listDates = driver.findElements(
				By.xpath("//div[@class='DayPicker-Month']/preceding-sibling::div/descendant::p[@class='fsw__date']"));

		// step 10: iterate all dates

How to scroll down using Actions class?
		-we perform scroll down by using moveToElement() method,
		- moveToElement() method is used mouse over on element without click.
		- first we have to find the scroll down element or object,
		- then we create object of Actions class by passing WebDriver instance
		- then we use moveToElement() method by passing WebElement instance then build() and perform() method

Scenario:
	- open a application
	- find the scroll down elements and scroll down


Step 1: up casting
		WebDriver driver = new ChromeDriver();

step 2: open a url
		driver.get("");

step 3: find the scroll  down element
		WebElement wb = driver.findElement(By.xpath(""));

step 4: create object of Actions class by passing WebDriver instance
		Actions act =new Actions(driver);

step 5: use moveToElement() method
		act.moveToElement(wb).build().perform();
