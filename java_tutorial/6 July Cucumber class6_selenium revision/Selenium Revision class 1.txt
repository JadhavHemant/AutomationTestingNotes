1) get() method
		-get() method is used to open a url in browser window
		- this method present inside the WebDriver interface
		- return type is void.
		- this method accept String type of arguments.

syntax:
		driver.get("");

step 1: up casting
			WebDriver driver =new ChromeDriver();

step 2: open a url
			driver.get("https://www.facebook.com")

-------------------------------------------------------------------------------------
navigate() method
	- navigate() method present inside the WebDriver interface.
	- return type is Navigation interface.
-------------------------------------------------------------------------------------
2) navigate().to() method
	- to() method is used to navigate from one url to another url as well as it used to open a new url
	- to()method is present inside the Navigation interface
	- return type of to() method
	- to() method accept the String as well as URL type values.
	
syntax:
		driver.navigate().to("https://www.facebook.com");

		Navigation abc = driver.navigate();
		abc.to("https://www.facebook.com");
-------------------------------------------------------------------------------------
diff between get() and navigate().to() method
	- get() method it will wait till all component to be loaded in webpage
	- navigate().to() method will not wait till all component to be loaded in webpage
	- get() method present in WebDriver interface
	-  navigate().to() method present inside the Navigation interface

	- get() method is accept the String type of args
	- navigate().to() method accept the String as well as URL type of args
-------------------------------------------------------------------------------------
3) navigate().back() method
	- it used to navigate one step back in browser history
	-  back() method present inside the Navigation interface
	- return type is void.

4) navigate().forward() method
	- it used to navigate one step forward in browser history
	-  forward() method present inside the Navigation interface
	- return type is void.


5) navigate().refresh() method
	- it used to refresh the web page
	-  refresh() method present inside the Navigation interface
	- return type is void.

package Tutorial1;

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;

public class Demo1 {

	public static void main(String[] args) throws InterruptedException {

		WebDriver driver = new ChromeDriver();

		driver.get("https://amazon.com");

		Thread.sleep(4000);

		driver.navigate().to("http://redbus.com");
		Thread.sleep(4000);

		driver.navigate().back();

		Thread.sleep(4000);

		driver.navigate().forward();

		Thread.sleep(4000);
		driver.navigate().refresh();
	}

}

6) maximize or minimize or fullScreen method

close() method
	- it used to close a focused or current window which is opened by selenium script.
	- this method is WebDriver interface.
	- return type is void

quit() method
	- used to close all windows.
	- this method is WebDriver interface.
	- return type is void


diff between close() method and quit() method?
	- close() method will not terminate the session
	- quit() method it will terminate the session
	- close() method is used to close focuses window
	- quit() method close all window


package Tutorial1;

import org.openqa.selenium.WebDriver;
import org.openqa.selenium.WindowType;
import org.openqa.selenium.chrome.ChromeDriver;

public class Demo2 {

	public static void main(String[] args) throws InterruptedException {

		WebDriver driver = new ChromeDriver();

		driver.get("https://amazon.com");
		Thread.sleep(4000);

		driver.switchTo().newWindow(WindowType.WINDOW);

		driver.get("https://redbus.com");
		Thread.sleep(4000);

		driver.switchTo().newWindow(WindowType.WINDOW);

		driver.get("https://facebook.com");
		Thread.sleep(4000);

		driver.close();

	}

}

package Tutorial1;

import org.openqa.selenium.Dimension;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;

public class Demo3 {

	public static void main(String[] args) throws InterruptedException {

		WebDriver driver = new ChromeDriver();

		Dimension d = new Dimension(1000, 200);
		driver.manage().window().setSize(d);

		Thread.sleep(4000);
		driver.get("https://amazon.com");
		Thread.sleep(4000);

	}

}

getTitle() method
	used to capture current browser window title
	- present inside WebDriver interface
	- return type is String

getCurrentUrl() method
	used to capture current browser window URL
	- present inside WebDriver interface
	- return type is String

package Tutorial1;

import org.openqa.selenium.Dimension;
import org.openqa.selenium.WebDriver;
import org.openqa.selenium.chrome.ChromeDriver;

public class Demo3 {

	public static void main(String[] args) throws InterruptedException {

		WebDriver driver = new ChromeDriver();

		Thread.sleep(4000);
		driver.get("https://amazon.com");

		String actualTitle = driver.getTitle();
		System.out.println(actualTitle);

		String actualUrl = driver.getCurrentUrl();
		System.out.println(actualUrl);
	}

}
